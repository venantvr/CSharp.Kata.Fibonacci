# CSharp.Kata.Fibonacci

Kata de programmation : implémentation de la suite de Fibonacci en TDD avec gestion des cas limites.

## Structure

- `Kata.Calculator/FibonacciCalculator.cs` — Calculateur de Fibonacci
- `Kata.Calculator/FibonacciOutOfRangeException.cs` — Exception pour valeurs hors limites
- `Kata.Fibonacci/FibonacciTest.cs` — Tests unitaires

## Stack
